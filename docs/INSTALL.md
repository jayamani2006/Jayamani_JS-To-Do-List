# Installation Guide

## For End Users (No Code Required)

If you just want to use the JS To-Do-List app, follow these simple steps:

1. **Download the App:**
   Go to the [Releases](https://github.com/jayamani2006/JS-To-Do-List/releases) page and download the latest `.exe` file (e.g., `JS-To-Do-List-v1.0.0-windows-x64.exe`).

2. **Run the App:**
   Double-click the downloaded `.exe` file. There is no installation process required—the app is fully portable!

3. **Enjoy:**
   The app will automatically create necessary folders (`data/` and `task_attachments/`) in the same directory as the executable to safely store your tasks and attachments.
