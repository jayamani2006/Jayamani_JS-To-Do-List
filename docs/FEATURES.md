# Features

## Core Functionality
- **Unlimited Folders:** Organize tasks cleanly into discrete lists.
- **Rich Task Information:** Set task name, due dates, due times, and descriptive notes.
- **Full Offline Mode:** Operates 100% locally with zero cloud dependencies.

## Advanced Attachments
- **File Support:** Attach images, MP3s, DOCX, XLSX, PDFs, or any file directly to a task.
- **Web Links:** Attach URLs to open web pages instantly.
- **Local Sandbox:** Attachments are copied to a local `task_attachments/` directory so they are never lost if the original file moves.

## UI & Visuals
- **Futuristic Neon Theme:** Dark mode default with cyan neon glowing borders.
- **Time Reference Color Coding:** Quickly assess the urgency and status of all tasks with color-coded feedback (Green, Yellow, Red, White).

## Automation
- **Midnight Watcher:** Automatically tracks and duplicates uncompleted tasks from Daily Habit and Health Care folders every midnight.
- **Auto-migrations:** Zero-hassle database upgrades for future versions.
