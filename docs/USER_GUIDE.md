# User Guide

Welcome to the JS To-Do-List User Guide!

## Main Dashboard

When you open the app, you will see a sidebar on the left and a task area on the right.

### Folders

You can create unlimited custom folders to categorize your tasks. At the bottom of the sidebar are special folders:
- **Daily Habit:** Tasks that repeat every single day.
- **Health Care:** Tasks related to your personal health.
- **Time Reference:** A special color-coded view to quickly assess task urgency.

### Adding Tasks

Click "Add Task" to create a new task. You can set:
- **Task Name**
- **Due Date & Time**
- **Notes**
- **Attachments:** Attach any file (image, PDF, MP3, etc.). The app makes a local copy so your attachments are always safe.
- **Links:** Attach web links directly to the task.

### Completing Tasks

When a task is complete, click the checkmark. You can optionally provide feedback:
- Perfect
- Good
- Average
- Poor
- Other (with custom comment)

## Time Reference View

This view is highly useful to check the status of all tasks:
- **Green:** Task is complete with positive feedback.
- **Yellow:** Task is complete with average feedback.
- **Red:** Task is overdue or terminated.
- **White:** Task is pending.

## Midnight Watcher

Tasks inside "Daily Habit" and "Health Care" are monitored by a background watcher. When the clock strikes midnight, incomplete tasks can be managed and tracked daily without manual recreation.
