# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-15
### Added
- Initial public release of JS To-Do-List.
- SQLite backend with automatic database migrations.
- Folder-based task organization.
- Support for rich file attachments (images, audio, documents).
- Time Reference view for color-coded task urgency.
- Daily habit tracking and "midnight watcher" task duplication.
- Compiled standalone executable via PyInstaller.
