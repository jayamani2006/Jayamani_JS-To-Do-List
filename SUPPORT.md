## 🐛 Bug Reports & Feature Requests
Please use our GitHub Issues page to report bugs or request features. 
Before opening a new issue, please check existing open issues to avoid duplicates.

* **Issue Tracker:** [Open an Issue](https://github.com/jayamani2006/Jayamani_JS-To-Do-List/issues)

When reporting a bug, please include:
1. Your Windows version (10/11)
2. Steps to reproduce the bug
3. Screenshots or videos (if applicable)
