# Roadmap

This document outlines the future vision and planned features for JS To-Do-List.

- [ ] Variable difficulty modes / Task prioritization levels
- [ ] Auditory feedback for task completion
- [ ] Cloud sync functionality for cross-device access
- [ ] Mobile companion app
- [ ] Cross-platform builds (macOS and Linux)
- [ ] Enhanced keyboard shortcut support
- [ ] Search functionality within attachments and task notes
